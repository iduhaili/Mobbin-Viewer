# Mobbin Viewer
![LOGO](https://github.com/A2coffee/Mobbin-Viewer/blob/main/public/mobbin-helper-icon-72.png?raw=true)

Mobbin Viewer 是一个面向 `mobbin.com` 的 Chrome 扩展，提供去模糊浏览、全屏查看、复制素材与批量下载能力。  

> [!WARNING]
> 此扩展仅用于教育目的。我不对任何滥用行为、违反服务条款或因使用本扩展而产生的任何后果负责。使用风险自负。

## Features

- `视觉还原` : 去除 Mobbin 页面素材的模糊遮罩，恢复正常浏览体验。
- `全屏预览` : 支持更直接地查看页面素材内容。
- `素材复制` : 提供便捷复制能力，减少手动提取成本。
- `批量下载` : 支持批量导出当前页面素材。

## Install

1. 打开 Chrome 的 `chrome://extensions`
2. 开启开发者模式并选择“加载已解压的扩展程序”

## Development

- 安装依赖: `pnpm install`
- 构建扩展: `pnpm build`
- 构建输出目录: `dist/`
